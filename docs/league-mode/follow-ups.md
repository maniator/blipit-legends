## How to use this doc

- **Adding to follow-ups:** any time the v0 planning PR or a phase implementation PR explicitly defers an item, add a row here with owner + blocking phase.
- **Closing items:** when a follow-up ships in a later PR, delete its row and reference the merged PR in that PR's commit message.
- **Quarterly review:** `@pm-agent` walks this list at each phase kickoff and confirms ownership / re-prioritizes blockers.
